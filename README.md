# Ai-FileManager
文档智能管家
使用python实现和langchain实现一个Ai文件管家的chatbot。
PDF文本提取：使用OCR大语言模型对PDF文件进行解析，提取文本内容。
聊天内容，文本内容总结并生成结构化信息：使用通用大语言模型，对用户的聊天内容和文本内容进行总结，生成结构化的信息，例如时间、地点、人物等。

用户通过与管家沟通有如下场景：

1. 用户向管家提交文件，让管家进行入库，管家先对文件进行拆分，然后将拆分后的文件返回给用户，让用户确认。 等用户最终确认拆分正确后，将文件入库，并且存入到RAG数据库。 入库以后如果是有效期的时间，自动识别，然后生成提醒内容给用户确认到期是否需要提醒。用户确认后，增加定时邮件提醒。
2. 用户向管家查询 之前存入文档的某个信息，管家通过RAG搜索找出用户需要查询的内容，然后把相关文档以及问题的答案返回给用户。
3. 支持dashboard展示，与当前用户，当前部门相关的整体的文档数据，包含即将过期和已经过期的文档。
